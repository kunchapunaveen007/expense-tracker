# Expense Tracker App

A simple **Expense Tracker** web application built with **Flask**, **SQLite**, and **JavaScript**.  
This app allows you to **add, edit, delete, and view expenses** from the browser with a clean and interactive UI.

---

## Features

- Add expenses with **title** and **amount**  
- View all expenses in a dynamic list  
- Edit existing expenses directly in the browser  
- Delete expenses with a click  
- RESTful API using **Flask** and **SQLAlchemy**

---

## Technologies Used

- Python 3  
- Flask  
- Flask-SQLAlchemy  
- JavaScript (Vanilla)  
- HTML / CSS  
- SQLite
